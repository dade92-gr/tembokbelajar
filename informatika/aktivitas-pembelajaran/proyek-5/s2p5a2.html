💻 BAGIAN E: IPO DI PROGRAM (KELOMPOK)

"Lihat bagaimana IPO diterapkan dalam program komputer!"
Apa itu Pseudocode?
Pseudocode adalah cara menuliskan logika program dengan bahasa yang mudah dipahami, mirip seperti instruksi sehari-hari. Di bawah ini adalah pseudocode program ATM. Tugas kalian: identifikasi mana yang termasuk Input, Proses, dan Output!
PROGRAM TarikTunaiATM

// DEKLARASI VARIABEL (Data yang akan digunakan)
saldo = 1000000 ← Data awal (simpanan)
pinUser = 123456 ← Data PIN yang benar

// ===== INPUT =====
TAMPILKAN "Masukkan kartu ATM"
TAMPILKAN "Masukkan PIN"
MASUKKAN pinInput ← Data dari user (PIN yang dimasukkan)

// ===== PROSES =====
IF pinInput == pinUser THEN
TAMPILKAN "Masukkan jumlah tarikan"
MASUKKAN jumlahTarik ← Data dari user (jumlah yang ingin ditarik)

IF jumlahTarik <= saldo THEN
saldo = saldo - jumlahTarik ← Proses pengurangan saldo

// ===== OUTPUT =====
TAMPILKAN "Uang keluar: " + jumlahTarik
TAMPILKAN "Saldo tersisa: " + saldo
TAMPILKAN "Cetak struk"
ELSE
TAMPILKAN "Saldo tidak mencukupi"
ENDIF
ELSE
TAMPILKAN "PIN salah"
ENDIF

END

Petunjuk: Perhatikan kode di atas. Bagian mana yang termasuk INPUT? PROSES? OUTPUT? Tuliskan jawabanmu di tabel di bawah!
