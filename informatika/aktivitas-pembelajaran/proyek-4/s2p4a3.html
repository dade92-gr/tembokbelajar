🔍 BAGIAN D: STUDI KASUS (KELOMPOK) – 20 menit

"Analisis peran sistem operasi dalam kasus nyata!"

Aktivitas 5: Studi Kasus 1 – Menjalankan Program

Kasus: Seorang user ingin menjalankan program Microsoft Word untuk mengetik dokumen.

  1. Bagaimana sistem operasi memuat program Word dari harddisk ke RAM?
  2. Bagaimana sistem operasi mengalokasikan memori untuk program Word?
  3. Bagaimana sistem operasi mengatur CPU untuk menjalankan program Word?
  4. Bagaimana sistem operasi menangani input dari keyboard saat user mengetik?
  5. Bagaimana sistem operasi menampilkan output ke layar monitor?

Aktivitas 6: Studi Kasus 2 – Multitasking

Kasus: Seorang user menjalankan beberapa program sekaligus: browser (5 tab), Spotify, WhatsApp, dan Microsoft Word.

  1. Bagaimana sistem operasi membagi waktu CPU untuk semua program yang berjalan?
  2. Apa yang terjadi jika memori (RAM) tidak cukup untuk menjalankan semua program?
  3. Apa yang dimaksud dengan swapping atau virtual memory?
  4. Bagaimana sistem operasi memastikan program tidak saling mengganggu?
  5. Mengapa komputer bisa terasa lambat jika terlalu banyak program berjalan?

Aktivitas 7: Studi Kasus 3 – Booting

Kasus: Proses menyalakan komputer dari kondisi mati hingga siap digunakan.

  1. Apa yang terjadi pertama kali saat tombol power ditekan?
  2. Apa peran BIOS/UEFI dalam proses booting?
  3. Bagaimana sistem operasi dimuat dari harddisk ke memori?
  4. Apa yang dimaksud dengan kernel sistem operasi?
  5. Mengapa proses booting membutuhkan waktu beberapa saat?
