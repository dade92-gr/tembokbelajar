📚 PETUNJUK PENGGUNAAN LKPD

Sebelum mengerjakan LKPD ini, pastikan kamu telah:
  1. Membaca materi "Pseudocode untuk Mesin Sederhana" dan "Implementasi HTML" yang tersedia di menu Daftar Materi pada website pembelajaran.
  2. Memahami model IPO dari Aktivitas 1 dan 2.
  3. Memahami konsep sistem operasi dari Aktivitas 3.
  4. Menyiapkan smartphone/laptop dengan HTML Editor.

🎯 TUJUAN AKTIVITAS 4

Setelah mengikuti aktivitas ini dan membaca materi yang tersedia, peserta didik diharapkan mampu:

  1. Menulis pseudocode untuk mesin sederhana berdasarkan model IPO.
  2. Mengimplementasikan pseudocode ke dalam program HTML.
  3. Menguji program yang telah dibuat.
  4. Menyempurnakan program berdasarkan hasil uji coba.
