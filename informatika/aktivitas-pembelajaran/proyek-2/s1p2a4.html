LEMBAR KERJA PESERTA DIDIK (LKPD) PROYEK 2 – AKTIVITAS 4

HTML DASAR DENGAN HTML EDITOR

Tujuan Pembelajaran:

TP 1.4: Menggunakan algoritma dan struktur data standar untuk menyelesaikan berbagai persoalan nyata.

TP 1.5: Menuliskan rancangan solusi program sederhana dalam bentuk pseudocode.

Identitas Kelompok

📚 PETUNJUK PENGGUNAAN LKPD

Sebelum mengerjakan LKPD ini, pastikan kamu telah:
  1. Membaca materi "Pseudocode Dasar" dan "HTML Dasar" yang tersedia di menu Daftar Materi pada website pembelajaran.
  2. Memahami konsep pseudocode sebagai jembatan menuju kode program.
  3. Menyiapkan smartphone/laptop dengan HTML Editor (bisa menggunakan website seperti codepen.io, jsfiddle.net, atau aplikasi HTML Editor online).
  4. Membawa hasil algoritma dari Aktivitas 3 untuk diimplementasikan.

🎯 TUJUAN AKTIVITAS 4

Setelah mengikuti aktivitas ini dan membaca materi yang tersedia, peserta didik diharapkan mampu:

  1. Menuliskan pseudocode untuk menyelesaikan masalah sederhana.
  2. Mengenal struktur dasar dokumen HTML.
  3. Membuat program sederhana menggunakan HTML Editor.
  4. Mengimplementasikan algoritma ke dalam bentuk program HTML.
