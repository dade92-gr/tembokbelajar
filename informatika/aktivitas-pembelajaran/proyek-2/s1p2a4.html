    // Proses

    // Output

END

🏗️ BAGIAN E: MEMBUAT PROGRAM SEDERHANA (KELOMPOK) – 20 menit

"Implementasikan pseudocode ke dalam HTML!"

Kegiatan 4: Program Kalkulator Sederhana

Instruksi: Buat program HTML sederhana untuk menghitung luas persegi panjang. Gunakan panduan berikut!

Pseudocode yang akan diimplementasikan:

PROGRAM LuasPersegiPanjang
    // Deklarasi variabel
    panjang, lebar, luas : integer

    // Input
    INPUT panjang
    INPUT lebar

    // Proses
    luas = panjang × lebar

    // Output
    OUTPUT "Luas persegi panjang adalah: " + luas
END

Kode HTML:

<!DOCTYPE html>
<html>
<head>
    <title>Kalkulator Luas Persegi Panjang</title>
</head>
<body>
    <h1>Program Menghitung Luas Persegi Panjang</h1>

    <form>
        <label>Panjang:</label>
        <input type="number" id="panjang"><br><br>

        <label>Lebar:</label>
        <input type="number" id="lebar"><br><br>

        <button type="button" onclick="hitungLuas()">Hitung Luas</button>
    </form>

    <p id="hasil"></p>

    <script>
        function hitungLuas() {
            var p = document.getElementById('panjang').value;
            var l = document.getElementById('lebar').value;
            var luas = p * l;
            document.getElementById('hasil').innerHTML = "Luas persegi panjang adalah: " + luas;
        }
    </script>
</body>
</html>

Tugas:

  1. Salin kode di atas ke HTML Editor dan jalankan!
  2. Uji coba dengan memasukkan panjang = 10, lebar = 5. Berapa hasilnya?
  3. Uji coba dengan angka lain. Apakah program berjalan dengan baik?
  4. Jika ada error, diskusikan dengan kelompok dan perbaiki!

Kegiatan 5: Program Menentukan Ganjil Genap

Instruksi: Buat program HTML untuk menentukan apakah suatu bilangan ganjil atau genap!

Pseudocode:

PROGRAM GanjilGenap

    INPUT bilangan

        OUTPUT bilangan + " adalah bilangan genap"
    ELSE
        OUTPUT bilangan + " adalah bilangan ganjil"
    ENDIF
END

Tugas: Tuliskan kode HTML berdasarkan pseudocode di atas!

<!DOCTYPE html>
<html>
<head>

    <form>
        <label>Masukkan bilangan:</label>
        <input type="number" id="bilangan"><br><br>
        <button type="button" onclick="cekGanjilGenap()">Cek</button>
    </form>

    <p id="hasil"></p>

    <script>
        function cekGanjilGenap() {
            // Tulis kode JavaScript di sini
            var bil = document.getElementById('bilangan').value;

            // Lengkapi kode berikut

                document.getElementById('hasil').innerHTML = bil + " adalah bilangan genap";

            }

Kegiatan 7: Program Kasir Mini

Instruksi: Buat program HTML sederhana untuk kasir mini yang dapat menghitung total belanja!

Kasus:

Seorang pembeli membeli 3 item:

  • Item 1: Harga Rp 15.000, jumlah 2
  • Item 2: Harga Rp 10.000, jumlah 3
  • Item 3: Harga Rp 20.000, jumlah 1

Tugas:

  1. Buat pseudocode untuk program kasir mini!
  2. Implementasikan ke dalam HTML!

Pseudocode:

PROGRAM KasirMini
    // Deklarasi variabel
    harga1, harga2, harga3 : integer
    jumlah1, jumlah2, jumlah3 : integer
    total1, total2, total3, totalSemua : integer

    // Input (dalam program nanti input dari user)
    harga1 = 15000, jumlah1 = 2
    harga2 = 10000, jumlah2 = 3
    harga3 = 20000, jumlah3 = 1

    // Proses
    total1 = harga1 × jumlah1
    total2 = harga2 × jumlah2
    total3 = harga3 × jumlah3
    totalSemua = total1 + total2 + total3

    // Output
    OUTPUT "Total belanja: Rp " + totalSemua
END
