Kegiatan 2: If-Else dengan Banyak Kondisi

Instruksi: Buat program yang dapat mengkonversi nilai angka menjadi nilai huruf dengan ketentuan:

  • A = 86 - 100
  • B = 71 - 85
  • C = 56 - 70
  • D = 41 - 55
  • E = 0 - 40

Pseudocode:

PROGRAM KonversiNilai
    // Deklarasi variabel
    nilai : integer

    // Input
    INPUT nilai

    // Proses dan Output
    IF nilai >= 86 AND nilai <= 100 THEN
        OUTPUT "Nilai A (Sangat Baik)"
    ELSE IF nilai >= 71 AND nilai <= 85 THEN
        OUTPUT "Nilai B (Baik)"
    ELSE IF nilai >= 56 AND nilai <= 70 THEN
        OUTPUT "Nilai C (Cukup)"
    ELSE IF nilai >= 41 AND nilai <= 55 THEN
        OUTPUT "Nilai D (Kurang)"
    ELSE IF nilai >= 0 AND nilai <= 40 THEN
        OUTPUT "Nilai E (Sangat Kurang)"
    ELSE
        OUTPUT "Nilai tidak valid"
    ENDIF
END

Kegiatan 5: Program Kasir dengan Diskon

Kasus: Sebuah toko memberikan diskon berdasarkan total belanja:

  • Total < Rp 50.000: Tidak dapat diskon
  • Total Rp 50.000 - Rp 100.000: Diskon 5%
  • Total Rp 100.000 - Rp 200.000: Diskon 10%
  • Total > Rp 200.000: Diskon 15%

Pseudocode:

PROGRAM KasirDiskon
    // Deklarasi variabel
    totalBelanja, diskon, totalBayar : integer

    // Input
    INPUT totalBelanja

    // Proses menentukan diskon
    IF totalBelanja < 50000 THEN
        diskon = 0
    ELSE IF totalBelanja <= 100000 THEN
        diskon = totalBelanja × 5/100
    ELSE IF totalBelanja <= 200000 THEN
        diskon = totalBelanja × 10/100
    ELSE
        diskon = totalBelanja × 15/100
    ENDIF

    // Hitung total bayar
    totalBayar = totalBelanja - diskon

    // Output
    OUTPUT "Total Belanja: Rp " + totalBelanja
    OUTPUT "Diskon: Rp " + diskon
    OUTPUT "Total Bayar: Rp " + totalBayar
END

Tugas: Buat program HTML berdasarkan pseudocode di atas!

<!DOCTYPE html>
<html>
<head>
    <title>Kasir dengan Diskon</title>
</head>

    <input type="number" id="totalBelanja">
    <button onclick="hitungDiskon()">Hitung</button>

    <p id="hasilTotal">Total Belanja: Rp </p>
    <p id="hasilDiskon">Diskon: Rp </p>

        function hitungDiskon() {

            var bayar = 0;

            // Tentukan diskon berdasarkan total belanja
            if (total < 50000) {
                diskon = 0;
            } else if (_________________________) {
                diskon = total * 5 / 100;
            } else if (_________________________) {
                diskon = total * 10 / 100;
            } else {
                diskon = total * 15 / 100;
            }

            // Hitung total bayar
            bayar = total - diskon;

            // Tampilkan hasil

Kegiatan 6: Program Tabel Perkalian

Kasus: Buat program yang dapat menampilkan tabel perkalian dari angka yang dimasukkan user.

Contoh: Jika user memasukkan angka 5, maka tampilkan:

1 × 5 = 5
2 × 5 = 10
3 × 5 = 15
...
10 × 5 = 50

Pseudocode:

PROGRAM TabelPerkalian
    // Deklarasi variabel
    angka, i : integer

    // Input
    INPUT angka

    // Proses dan Output
    FOR i = 1 TO 10
        OUTPUT i + " × " + angka + " = " + (i × angka)
    ENDFOR
END

Tugas: Buat program HTML berdasarkan pseudocode di atas!

<!DOCTYPE html>
<html>
<head>
    <title>Tabel Perkalian</title>
</head>
<body>
    <h1>Program Tabel Perkalian</h1>

    <label>Masukkan angka:</label>
    <input type="number" id="angka">
    <button onclick="tampilkanPerkalian()">Tampilkan</button>

    <p id="hasil"></p>

    <script>
        function tampilkanPerkalian() {
            var a = parseInt(document.getElementById('angka').value);
            var teks = "";

            // Gunakan perulangan for untuk menampilkan 1 sampai 10
            for (var i = 1; i <= 10; i++) {
                teks = teks + i + " × " + a + " = " + (i * a) + "<br>";
            }

            document.getElementById('hasil').innerHTML = teks;
        }
    </script>
</body>
</html>

Pengembangan:

  1. Ubah program agar user bisa menentukan batas perkalian (misal sampai 20)!
  2. Tambahkan input untuk batas bawah dan batas atas!

Kegiatan 7: Program Simulasi Antrean Kantin

Kasus: Sebuah kantin memiliki aturan:

  • Setiap siswa dapat membeli maksimal 3 item
  • Harga item: Rp 5.000 per item
  • Jika siswa membeli lebih dari 3 item, tampilkan pesan "Maksimal 3 item per siswa"

Pseudocode:

PROGRAM AntreanKantin
    // Deklarasi variabel
    jumlahItem, totalHarga : integer
    ulang : char = 'y'

    // Proses dengan perulangan
    WHILE ulang == 'y' OR ulang == 'Y'
        INPUT jumlahItem

        IF jumlahItem <= 3 THEN
            totalHarga = jumlahItem × 5000
            OUTPUT "Total harga: Rp " + totalHarga
        ELSE
            OUTPUT "Maksimal 3 item per siswa"
        ENDIF

        OUTPUT "Ada siswa lagi? (y/n): "
        INPUT ulang
    ENDWHILE

    OUTPUT "Terima kasih"
END

Tugas: Buat program HTML berdasarkan pseudocode di atas! (Gunakan prompt untuk input berulang)

<!DOCTYPE html>
<html>
<head>
    <title>Simulasi Antrean Kantin</title>
</head>
<body>
    <h1>Simulasi Antrean Kantin</h1>

    <button onclick="mulaiAntrean()">Mulai Antrean</button>

    <p id="hasil"></p>

    <script>
        function mulaiAntrean() {
            var teks = "";
            var ulang = "y";

            while (ulang == "y" || ulang == "Y") {
                var jumlah = prompt("Masukkan jumlah item yang dibeli:");
                jumlah = parseInt(jumlah);

                if (jumlah <= 3) {
                    var total = jumlah * 5000;
                    teks = teks + "Siswa membeli " + jumlah + " item, total Rp " + total + "<br>";
                } else {
                    teks = teks + "Siswa mencoba membeli " + jumlah + " item (TIDAK BOLEH - maksimal 3)<br>";
                }

                ulang = prompt("Ada siswa lagi? (y/n):");
            }

            teks = teks + "<br><strong>Antrean selesai. Terima kasih!</strong>";
            document.getElementById('hasil').innerHTML = teks;
        }
    </script>
</body>
</html>
