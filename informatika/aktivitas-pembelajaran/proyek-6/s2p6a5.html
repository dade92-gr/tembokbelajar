🔀 BAGIAN B: STRUKTUR KONDISI (IF-ELSE) – 20 menit

"Program bisa mengambil keputusan!"

Kegiatan 1: Memahami If-Else

// Program Menentukan Lulus/Tidak
let n = prompt("Masukkan nilai:");
if (n >= 75) {
    alert("Lulus");
} else {
    alert("Tidak Lulus");
}
NoPertanyaanJawaban
1Jika input nilai = 80, apa outputnya? Jika input nilai = 70, apa outputnya?
2Apa fungsi dari kondisi if (n >= 75) pada program di atas?
3Jika nilai yang dimasukkan adalah 100, apa outputnya? Jelaskan!

Kegiatan 2: If-Else dengan Banyak Kondisi (Grading Nilai)

// Program Grading Nilai
let n = prompt("Masukkan nilai:");
if (n >= 90) { alert("Nilai A (Sangat Baik)"); }
else if (n >= 75) { alert("Nilai B (Baik)"); }
else if (n >= 60) { alert("Nilai C (Cukup)"); }
else if (n >= 40) { alert("Nilai D (Kurang)"); }
else { alert("Nilai E (Sangat Kurang)"); }

🔁 BAGIAN C: PERULANGAN (LOOPING) – 20 menit

"Program bisa melakukan tugas berulang!"

Kegiatan 3: Perulangan For

// Menampilkan angka 1-10
for (let i = 1; i <= 10; i++) {
    document.write(i + " ");
}
NoPertanyaanJawaban
1Apa output dari program di atas?
2Jika i <= 10 diubah menjadi i <= 5, apa outputnya?
3Buat program untuk menampilkan angka 10 sampai 1 menggunakan for!

Kegiatan 4: Perulangan While

// Menampilkan angka genap 2-20
let i = 2;
while (i <= 20) {
    document.write(i + " ");
    i = i + 2;
}
NoPertanyaanJawaban
1Apa output dari program di atas?
2Jika i = i + 2 diubah menjadi i = i + 3, apa outputnya?
3Buat program untuk menampilkan angka ganjil 1-19 menggunakan while!
