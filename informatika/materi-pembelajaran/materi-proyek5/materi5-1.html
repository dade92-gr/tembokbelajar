A. Apa Itu Model IPO?

Model Input-Process-Output (IPO) adalah kerangka kerja konseptual yang menggambarkan bagaimana suatu sistem memproses data atau masukan untuk menghasilkan keluaran. Model ini merupakan fondasi dasar dari hampir semua sistem komputasi.

Donald Knuth: "Model IPO adalah representasi sederhana dari cara kerja sistem komputer yang menerima masukan, memprosesnya, dan menghasilkan keluaran."
INPUT
Data mentah masuk ke sistem
PROSES
Pengolahan data
OUTPUT
Hasil olahan keluar sistem
Ilustrasi Sederhana:
🍳 Memasak Mie Instan:
• INPUT: Mie instan, air, kompor, bumbu
• PROSES: Rebus air, masukkan mie, masak 3 menit, campur bumbu
• OUTPUT: Mie instan matang siap santap

B. Tiga Komponen Utama Model IPO

Klik pada setiap komponen untuk memperjelas tampilan:

INPUT (Masukan)
Data atau perintah yang dimasukkan ke dalam sistem untuk diproses. Contoh: keyboard, mouse, sensor, mikrofon, kamera.
PROSES (Pengolahan)
Langkah-langkah pengolahan data: perhitungan, perbandingan, pengambilan keputusan, penyimpanan, pengambilan data.
OUTPUT (Keluaran)
Hasil dari proses yang dikeluarkan sistem. Contoh: monitor, speaker, printer, lampu, file data.

C. Contoh IPO dalam Kehidupan Sehari-Hari

Klik pada setiap contoh untuk memperjelas tampilan:

Mesin ATM
Input: Kartu ATM, PIN, angka nominal
Proses: Verifikasi PIN, cek saldo, kurangi saldo
Output: Uang keluar, struk, layar informasi
Kalkulator
Input: Angka, operator (+, -, ×, ÷)
Proses: Perhitungan matematika sesuai operator
Output: Hasil perhitungan di layar
Mesin Penjual Otomatis
Input: Uang koin/uang kertas, pilihan produk
Proses: Verifikasi uang, hitung kembalian
Output: Produk keluar, uang kembalian
Sistem Absensi Sekolah
Input: Sidik jari atau kartu RFID, waktu masuk
Proses: Verifikasi identitas, catat waktu
Output: Notifikasi absensi, laporan kehadiran

E. IPO dalam Sistem Komputer

Sistem Komputer:
• Input Device: Keyboard, mouse, scanner, mikrofon
• Process: CPU (Central Processing Unit) - melakukan perhitungan dan pengolahan data
• Memory: RAM (menyimpan data sementara), Storage (menyimpan data permanen)
• Output Device: Monitor, speaker, printer

F. IPO dalam Pemrograman Sederhana

📝 Program Luas Persegi Panjang:
// INPUT
var panjang = document.getElementById('panjang').value;
var lebar = document.getElementById('lebar').value;

// PROSES
var luas = panjang * lebar;

// OUTPUT
document.getElementById('hasil').innerHTML = "Luas = " + luas;
Analisis IPO:
• Input: Nilai panjang dan lebar dari pengguna
• Proses: Perhitungan luas = panjang × lebar
• Output: Tampilan hasil perhitungan di layar
