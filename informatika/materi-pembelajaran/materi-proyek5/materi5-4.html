A. Apa Itu Pseudocode?

Pseudocode adalah cara menuliskan algoritma menggunakan bahasa yang mirip dengan bahasa pemrograman, tetapi masih mudah dipahami oleh manusia. Pseudocode membantu kita merancang logika program sebelum diimplementasikan ke dalam kode.

Pseudocode adalah jembatan antara ide dan kode program. Tulis pseudocode dengan jelas agar mudah diterjemahkan ke bahasa pemrograman apapun.

Mengapa Pseudocode Penting?

📌 Manfaat Pseudocode:
• Perencanaan - Merancang logika sebelum membuat program
• Visualisasi - Melihat alur input → proses → output
• Komunikasi - Memudahkan diskusi dalam kelompok
• Dokumentasi - Menjadi catatan cara kerja program
• Debugging - Memudahkan menemukan kesalahan logika

B. Kata Kunci dalam Pseudocode

Klik pada setiap kata kunci untuk memperjelas tampilan:

START / END
Menandai awal dan akhir algoritma
INPUT / READ
Menerima masukan dari pengguna
OUTPUT / PRINT
Menampilkan hasil
IF / ELSE / ENDIF
Pengambilan keputusan (kondisi)
← atau =
Memberi nilai pada variabel (assignment)

C. Struktur Pseudocode

PROGRAM NamaProgram DEKLARASI // Variabel yang dibutuhkan namaVariabel1 : tipeData namaVariabel2 : tipeData ALGORITMA START // ===== INPUT ===== INPUT data1 INPUT data2 // ===== PROSES ===== // Logika perhitungan, kondisi, perulangan // ===== OUTPUT ===== OUTPUT hasil END END PROGRAM

D. Contoh Pseudocode untuk Berbagai Mesin

Klik pada setiap mesin untuk memperjelas tampilan:

Kalkulator Sederhana
PROGRAM KalkulatorSederhana DEKLARASI a, b : integer hasil : integer operator : char ALGORITMA START OUTPUT "Masukkan angka pertama: " INPUT a OUTPUT "Masukkan operator (+, -, *, /): " INPUT operator OUTPUT "Masukkan angka kedua: " INPUT b IF operator = '+' THEN hasil ← a + b ELSE IF operator = '-' THEN hasil ← a - b ELSE IF operator = '*' THEN hasil ← a * b ELSE IF operator = '/' THEN hasil ← a / b ENDIF OUTPUT "Hasil: " + hasil END END PROGRAM
ATM Mini
PROGRAM ATM_Mini DEKLARASI saldo : integer ← 1000000 pin : integer ← 1234 inputPin : integer pilihan : integer jumlahTarik : integer ALGORITMA START OUTPUT "Masukkan PIN: " INPUT inputPin IF inputPin = pin THEN REPEAT OUTPUT "=== MENU ATM ===" OUTPUT "1. Cek Saldo" OUTPUT "2. Tarik Tunai" OUTPUT "3. Keluar" INPUT pilihan IF pilihan = 1 THEN OUTPUT "Saldo Anda: " + saldo ELSE IF pilihan = 2 THEN OUTPUT "Masukkan jumlah tarik: " INPUT jumlahTarik IF jumlahTarik <= saldo THEN saldo ← saldo - jumlahTarik OUTPUT "Tarik tunai berhasil" OUTPUT "Saldo tersisa: " + saldo ELSE OUTPUT "Saldo tidak mencukupi" ENDIF ENDIF UNTIL pilihan = 3 ELSE OUTPUT "PIN salah" ENDIF END END PROGRAM
Konversi Suhu
PROGRAM KonversiSuhu DEKLARASI celcius, fahrenheit : float pilihan : integer ALGORITMA START OUTPUT "=== KONVERSI SUHU ===" OUTPUT "1. Celcius ke Fahrenheit" OUTPUT "2. Fahrenheit ke Celcius" INPUT pilihan IF pilihan = 1 THEN OUTPUT "Masukkan suhu Celcius: " INPUT celcius fahrenheit ← (celcius * 9/5) + 32 OUTPUT celcius + "°C = " + fahrenheit + "°F" ELSE IF pilihan = 2 THEN OUTPUT "Masukkan suhu Fahrenheit: " INPUT fahrenheit celcius ← (fahrenheit - 32) * 5/9 OUTPUT fahrenheit + "°F = " + celcius + "°C" ENDIF END END PROGRAM
Penghitung Luas
PROGRAM PenghitungLuas DEKLARASI panjang, lebar, sisi, alas, tinggi : float luas : float pilihan : integer ALGORITMA START OUTPUT "=== HITUNG LUAS ===" OUTPUT "1. Persegi Panjang" OUTPUT "2. Segitiga" OUTPUT "3. Persegi" INPUT pilihan IF pilihan = 1 THEN OUTPUT "Masukkan panjang: " INPUT panjang OUTPUT "Masukkan lebar: " INPUT lebar luas ← panjang * lebar OUTPUT "Luas Persegi Panjang: " + luas ELSE IF pilihan = 2 THEN OUTPUT "Masukkan alas: " INPUT alas OUTPUT "Masukkan tinggi: " INPUT tinggi luas ← (alas * tinggi) / 2 OUTPUT "Luas Segitiga: " + luas ELSE IF pilihan = 3 THEN OUTPUT "Masukkan sisi: " INPUT sisi luas ← sisi * sisi OUTPUT "Luas Persegi: " + luas ENDIF END END PROGRAM

F. Implementasi HTML

Berikut adalah struktur dasar HTML untuk mengimplementasikan pseudocode:

Struktur Dasar HTML Program

<!-- Struktur Dasar HTML untuk Program --> <!DOCTYPE html> <html> <head> <title>Nama Program</title> <style> /* CSS untuk tampilan */ </style> </head> <body> <!-- ===== INPUT ===== --> <label>Masukkan data: </label> <input type="text" id="input1"> <button onclick="proses()">Hitung</button> <!-- ===== OUTPUT ===== --> <p id="hasil"></p> <!-- ===== PROSES ===== --> <script> function proses() { // Ambil data dari input var data = document.getElementById('input1').value; // Lakukan perhitungan var hasil = data * 2; // Tampilkan output document.getElementById('hasil').innerHTML = "Hasil: " + hasil; } </script> </body> </html>

G. Uji Coba dan Debugging

Tips Debugging (Mencari dan Memperbaiki Kesalahan):
• Baca Pesan Error - Pesan error memberitahu lokasi dan jenis kesalahan
• Console.log() - Gunakan console.log() untuk melihat nilai variabel
• Uji dengan Data Sederhana - Uji dengan angka kecil yang mudah dihitung manual
• Periksa Tanda Baca - Periksa apakah ada tanda kutip, kurung, atau titik koma yang hilang
• Debug Bersama - Diskusikan dengan kelompok, kadang orang lain melihat kesalahan yang terlewat
