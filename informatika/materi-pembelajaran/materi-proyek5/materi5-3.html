A. Apa Itu Sistem Operasi?

Sistem Operasi (Operating System) adalah perangkat lunak sistem yang bertindak sebagai perantara antara pengguna, perangkat lunak aplikasi, dan perangkat keras komputer. Sistem operasi mengelola semua sumber daya komputer dan menyediakan layanan umum untuk program-program aplikasi.

Abraham Silberschatz: "Sistem operasi adalah program yang bertindak sebagai perantara antara pengguna komputer dan perangkat keras komputer."
👤 PENGGUNA
📱 APLIKASI (Browser, Game, Office)
⚙️ SISTEM OPERASI
🖥️ PERANGKAT KERAS (CPU, RAM, Harddisk)

B. Fungsi Utama Sistem Operasi

Klik pada setiap fungsi untuk memperjelas tampilan:

Manajemen Proses
Membuat/menghapus proses, menjadwalkan proses di CPU, sinkronisasi antar proses. Contoh: menjalankan browser, Word, dan game secara bersamaan.
Manajemen Memori
Melacak penggunaan memori, mengalokasikan ruang untuk proses, membebaskan memori. Contoh: membuka banyak aplikasi tanpa mengurangi kinerja.
Manajemen File
Membuat/menghapus file dan direktori, mengatur akses file. Contoh: menyimpan dokumen Word di folder Documents.
Manajemen I/O
Mengelola driver perangkat, buffering data, menangani interupsi. Contoh: menekan tombol keyboard, menggerakkan mouse.
Manajemen Keamanan
Autentikasi (login), otorisasi akses, enkripsi data. Contoh: login dengan password, pengaturan izin file.
Manajemen Jaringan
Protokol jaringan (TCP/IP), routing, keamanan jaringan. Contoh: browsing internet, streaming video.

D. Hubungan Sistem Operasi dengan Arsitektur Von Neumann

Klik pada setiap komponen untuk memperjelas tampilan:

CPU
Peran OS: Menjadwalkan proses, mengalokasikan waktu CPU, menangani interupsi. OS memastikan setiap program mendapat giliran menggunakan CPU secara bergiliran.
Memori (RAM)
Peran OS: Mengalokasikan ruang memori untuk setiap program, mengelola memori virtual, melindungi memori antar program. OS memastikan program tidak saling mengganggu di memori.
Input Device
Peran OS: Menyediakan driver untuk perangkat input, buffering data, menangani interupsi dari keyboard, mouse, scanner.
Output Device
Peran OS: Menyediakan driver untuk perangkat output, mengelola antrean cetak (spooling), menampilkan informasi ke monitor.
Storage
Peran OS: Manajemen file sistem, mengatur alokasi ruang di harddisk/SSD, caching data, menyimpan data secara permanen.

E. Peran Sistem Operasi dalam Proses Komputasi

Klik pada setiap tahap untuk memperjelas tampilan:

1
Saat Komputer Dinyalakan (Booting)
BIOS/UEFI menjalankan POST, mencari boot loader, memuat kernel OS ke memori
2
Saat Menjalankan Program
OS mengalokasikan memori, memuat program ke RAM, menjadwalkan proses di CPU
3
Saat Multitasking
OS membagi waktu CPU untuk banyak program, mengelola memori agar semua program berjalan
4
Saat Mengakses File
OS memeriksa izin akses, mencari lokasi file, membaca/menulis data melalui driver

F. Studi Kasus: Menjalankan Program Microsoft Word

📝 Peran OS Saat Menjalankan Microsoft Word:
1. Input: User mengklik ikon Word → OS menerima perintah
2. Storage: OS mencari file program Word di harddisk
3. Memori: OS mengalokasikan ruang di RAM untuk Word
4. Proses: OS memuat kode program dari harddisk ke RAM
5. CPU: OS membuat proses baru dan menjadwalkannya di CPU
6. Output: OS menampilkan jendela Word di monitor
7. I/O: Saat user mengetik, OS menangani input dari keyboard
8. File: Saat save, OS menyimpan file ke harddisk

G. Studi Kasus: Multitasking

Saat Menjalankan Banyak Program:
• Time Sharing: OS membagi waktu CPU secara bergiliran (millidetik)
• Memory Management: OS mengalokasikan RAM untuk setiap program
• Jika RAM Penuh: OS menggunakan memori virtual (harddisk sebagai RAM tambahan)
• Komputer Lambat: Terjadi karena CPU dan RAM kelebihan beban
Ingat! Sistem operasi adalah "jembatan" yang memungkinkan kita menggunakan komputer. Tanpa OS, komputer hanya kumpulan komponen elektronik yang tidak berguna!
