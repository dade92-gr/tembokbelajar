A. Pseudocode untuk Mesin Sederhana

Pseudocode adalah cara menuliskan algoritma menggunakan bahasa yang mirip dengan bahasa pemrograman, tetapi masih mudah dipahami oleh manusia. Untuk merancang mesin sederhana, pseudocode membantu kita memikirkan logika kerja mesin sebelum diimplementasikan ke dalam kode program.

📌 Mengapa Pseudocode Penting untuk Mesin Sederhana?
• Memudahkan Perencanaan - Merancang logika mesin sebelum membuat program
• Visualisasi Alur - Membantu melihat bagaimana input diproses menjadi output
• Identifikasi Kebutuhan - Memastikan semua kebutuhan mesin terpenuhi
• Komunikasi Tim - Memudahkan diskusi dalam kelompok
• Dokumentasi - Menjadi catatan cara kerja mesin

B. Struktur Pseudocode untuk Mesin Sederhana

PROGRAM NamaMesin DEKLARASI // Variabel yang dibutuhkan namaVariabel1 : tipeData namaVariabel2 : tipeData // ... variabel lainnya ALGORITMA START // ===== INPUT ===== INPUT data1 INPUT data2 // ===== PROSES ===== // Logika mesin, perhitungan, kondisi, perulangan // ===== OUTPUT ===== OUTPUT hasil END END PROGRAM

C. Kata Kunci dalam Pseudocode Mesin

Klik pada setiap kata kunci untuk memperjelas tampilan:

START / END
Menandai awal dan akhir algoritma
INPUT / READ
Menerima masukan dari pengguna
OUTPUT / PRINT
Menampilkan hasil
IF / ELSE / ENDIF
Pengambilan keputusan (kondisi)
FOR / ENDFOR
Perulangan dengan jumlah tetap
WHILE / ENDWHILE
Perulangan berdasarkan kondisi
← atau =
Memberi nilai pada variabel (assignment)

D. Contoh Pseudocode untuk Berbagai Mesin Sederhana

Klik pada setiap mesin untuk memperjelas tampilan:

Mesin ATM Mini
PROGRAM MesinATM_Mini

DEKLARASI
    saldo : float ← 1000000
    pin : integer ← 1234
    inputPin : integer
    pilihan : integer
    jumlahTarik : float

ALGORITMA
    START
        OUTPUT "Masukkan PIN: "
        INPUT inputPin
        IF inputPin = pin THEN
            ... [menu utama] ...
        ELSE
            OUTPUT "PIN salah"
        ENDIF
    END
END PROGRAM
Mesin Kasir Mini
PROGRAM MesinKasir_Mini

DEKLARASI
    harga1, harga2, harga3 : float
    jumlah1, jumlah2, jumlah3 : integer
    total1, total2, total3 : float
    totalSemua, diskon, bayar : float

ALGORITMA
    START
        // INPUT data barang
        INPUT harga1, jumlah1
        ...
        // PROSES perhitungan
        total1 ← harga1 * jumlah1
        totalSemua ← total1 + total2 + total3
        // Hitung diskon
        IF totalSemua >= 200000 THEN
            diskon ← totalSemua * 0.15
        ENDIF
        // OUTPUT hasil
        OUTPUT totalSemua, diskon, bayar
    END
END PROGRAM
Mesin Antrean
PROGRAM MesinAntrean

DEKLARASI
    nomorAntrean : integer ← 0
    nomorDipanggil : integer ← 0
    pilihan : integer
    ulang : char

ALGORITMA
    START
        REPEAT
            OUTPUT "=== SISTEM ANTREAN ==="
            OUTPUT "1. Ambil nomor"
            OUTPUT "2. Panggil nomor"
            OUTPUT "3. Lihat antrean"
            OUTPUT "4. Keluar"
            INPUT pilihan
            IF pilihan = 1 THEN ...
            INPUT ulang
        UNTIL ulang ≠ 'y'
    END
END PROGRAM
Mesin Vending
PROGRAM MesinVending

DEKLARASI
    minuman : array[1..5] of string
    harga : array[1..5] of float
    stok : array[1..5] of integer
    pilihan : integer
    uang : float

ALGORITMA
    START
        // Inisialisasi data
        minuman[1] ← "Air Mineral"
        harga[1] ← 5000
        ...
        // Tampilkan menu
        FOR i = 1 TO 5
            OUTPUT i + ". " + minuman[i]
        ENDFOR
        INPUT pilihan
        IF stok[pilihan] > 0 THEN ...
    END
END PROGRAM

G. Tips Menulis Pseudocode untuk Mesin

📝 Tips Penting:
1. Gunakan Bahasa Sederhana - Hindari istilah teknis yang rumit
2. Ikuti Model IPO - Pisahkan bagian input, proses, output
3. Beri Nama Variabel Jelas - Gunakan nama yang deskriptif (totalBayar, bukan tb)
4. Gunakan Indentasi - Buat struktur rapi agar mudah dibaca
5. Uji dengan Contoh - Coba jalankan secara manual dengan contoh angka
6. Perhatikan Kondisi Batas - Pikirkan apa yang terjadi jika input tidak valid
7. Tambahkan Komentar - Jelaskan bagian yang rumit dengan // komentar

H. Rangkuman

  • Pseudocode untuk mesin sederhana adalah cara merancang logika mesin sebelum diimplementasikan ke dalam program.
  • Struktur pseudocode terdiri dari deklarasi variabel, bagian input, proses, dan output.
  • Kata kunci penting: IF-ELSE untuk kondisi, FOR-WHILE untuk perulangan, INPUT-OUTPUT untuk I/O.
  • Model IPO membantu memisahkan bagian input, proses, dan output dalam pseudocode.
  • Contoh mesin: ATM, kasir, antrean, vending, mesin cuci, penukar uang.
  • Pseudocode memudahkan perencanaan, dokumentasi, dan komunikasi tim sebelum coding.
Ingat! Pseudocode adalah jembatan antara ide dan kode program. Tulis pseudocode dengan jelas agar mudah diterjemahkan ke bahasa pemrograman apapun!
