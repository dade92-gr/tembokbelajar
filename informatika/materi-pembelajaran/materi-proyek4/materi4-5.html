Implementasi HTML untuk Mesin Sederhana

Proyek 4: Rancang Mesin Pintar Mini - Pertemuan 4 4 JP TP 1.4

🎯 Tujuan Pembelajaran

Mengimplementasikan pseudocode mesin sederhana ke dalam program HTML
Membuat antarmuka pengguna (UI) untuk mesin sederhana
Menulis kode JavaScript untuk menjalankan logika mesin
Menguji program dan memperbaiki kesalahan (debugging)
Menyajikan program mesin sederhana yang siap digunakan

A. Dari Pseudocode ke Program HTML

Setelah menulis pseudocode untuk mesin sederhana, langkah selanjutnya adalah mengimplementasikannya ke dalam program HTML. Program HTML akan terdiri dari tiga bagian utama: HTML (struktur antarmuka), CSS (tampilan), dan JavaScript (logika mesin).

PSEUDOCODE
Rancangan algoritma
HTML + CSS
Struktur & Tampilan
JAVASCRIPT
Logika mesin
PROGRAM SIAP
Mesin siap digunakan

B. Struktur Dasar Program HTML untuk Mesin

Klik pada setiap bagian untuk memperjelas tampilan:

HTML (Struktur)
<div>
  <label>Input:</label>
  <input type="number" id="input1">
  <button onclick="proses()">Jalankan</button>
  <div id="hasil"></div>
</div>
CSS (Tampilan)
body {
  font-family: Arial;
  background: #f0f0f0;
}
button {
  background: #0066cc;
  color: white;
}
JavaScript (Logika)
function proses() {
  var input = document.getElementById('input1').value;
  var hasil = input * 2;
  document.getElementById('hasil').innerHTML = hasil;
}

D. Jenis Error dan Cara Debugging

Klik pada setiap jenis error untuk memperjelas tampilan:

Syntax Error
Salah penulisan kode. Cek typo, kurung, titik koma. Contoh: lupa tanda kurung tutup.
Runtime Error
Error saat program berjalan. Contoh: membagi dengan nol, memanggil variabel belum didefinisikan.
Logic Error
Program jalan tapi hasil salah. Contoh: rumus luas = panjang + lebar (harusnya dikali).
Input Error
User memasukkan data tidak valid. Contoh: input huruf di field angka, input kosong.
Tools Debugging:
• console.log() - Melihat nilai variabel di konsol browser
• alert() - Notifikasi sederhana untuk pengecekan
• Browser DevTools (F12) - Melihat error dan inspect kode
• debugger - Menghentikan eksekusi di titik tertentu

4

Gunakan Nama Variabel Jelas

6
Tambahkan Komentar

F. Rangkuman

  • Implementasi HTML adalah proses mengubah pseudocode menjadi program sungguhan.
  • Struktur program terdiri dari HTML (struktur), CSS (tampilan), JavaScript (logika).
  • Setiap mesin memiliki antarmuka dan logika yang berbeda sesuai kebutuhannya.
  • Contoh mesin: ATM, kasir, antrean, vending - semuanya bisa diimplementasikan dengan HTML/JavaScript.
  • Tips penting: mulai dari pseudocode, uji setiap fungsi, handle input tidak valid.
  • Debugging adalah proses mencari dan memperbaiki error dalam program.
Ingat! Program yang baik tidak hanya berjalan saat input benar, tapi juga harus bisa menangani input yang salah dengan memberikan pesan yang jelas!
