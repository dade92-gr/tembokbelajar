A. Apa Itu Sistem Operasi?

Sistem Operasi (Operating System) adalah perangkat lunak sistem yang bertindak sebagai perantara antara pengguna, perangkat lunak aplikasi, dan perangkat keras komputer. Sistem operasi mengelola semua sumber daya komputer dan menyediakan layanan umum untuk program-program aplikasi.

Abraham Silberschatz: "Sistem operasi adalah program yang bertindak sebagai perantara antara pengguna komputer dan perangkat keras komputer."
👤 PENGGUNA
📱 APLIKASI (Browser, Game, Office)
⚙️ SISTEM OPERASI
🖥️ PERANGKAT KERAS (CPU, RAM, Harddisk)
📌 Tujuan Sistem Operasi:
• Kenyamanan: Membuat penggunaan komputer menjadi lebih mudah
• Efisiensi: Mengelola sumber daya komputer secara efisien
• Kemampuan Berkembang: Memungkinkan pengembangan program baru
• Abstraksi: Menyembunyikan kerumitan perangkat keras
• Keamanan: Melindungi data dari akses tidak sah

B. Fungsi Utama Sistem Operasi

Klik pada setiap fungsi untuk memperjelas tampilan:

Manajemen Proses
Membuat/menghapus proses, menjadwalkan proses di CPU, sinkronisasi antar proses, penanganan deadlock.
Manajemen Memori
Melacak penggunaan memori, mengalokasikan ruang untuk proses, membebaskan memori, manajemen memori virtual.
Manajemen File
Membuat/menghapus file dan direktori, mengatur akses file, mengelola penyimpanan di harddisk/SSD.
Manajemen I/O
Mengelola driver perangkat, buffering data, spooling output, menangani interupsi dari perangkat.
Manajemen Keamanan
Autentikasi (login), otorisasi akses, enkripsi data, audit aktivitas pengguna.
Manajemen Jaringan
Protokol jaringan (TCP/IP), routing, keamanan jaringan, berbagi sumber daya (file/printer sharing).

D. Peran Sistem Operasi dalam Proses Komputasi

Klik pada setiap tahap untuk memperjelas tampilan:

1
Saat Komputer Dinyalakan (Booting) - BIOS/UEFI menjalankan POST, mencari boot loader, memuat kernel OS ke memori
2
Saat Menjalankan Program - Mengalokasikan memori, memuat program ke RAM, menjadwalkan proses di CPU
3
Saat Mengakses File - Memeriksa izin akses, mencari lokasi file, membaca/menulis data melalui driver

E. Hubungan Sistem Operasi dengan Arsitektur Von Neumann

Peran OS dalam Mengelola Komponen Von Neumann:
• CPU: Menjadwalkan proses, mengalokasikan waktu CPU, menangani interupsi
• Memori (RAM): Mengalokasikan ruang memori, manajemen memori virtual, proteksi memori
• Input Device: Menyediakan driver, buffering, menangani interupsi
• Output Device: Menyediakan driver, spooling, menangani antrean output
• Storage: Manajemen file sistem, alokasi ruang, caching
📝 Contoh: Peran OS Saat Menjalankan Browser Chrome
1. Pengguna klik ikon Chrome → OS menerima perintah
2. OS mencari file program Chrome di harddisk
3. OS mengalokasikan memori untuk Chrome
4. OS memuat kode program dari harddisk ke RAM
5. OS membuat proses baru dan menjadwalkannya di CPU
6. Chrome berjalan, OS menampilkan jendela
7. Saat Chrome membutuhkan internet, OS mengelola koneksi jaringan
8. Saat ditutup, OS membersihkan memori yang digunakan

F. Rangkuman

  • Sistem operasi adalah perangkat lunak yang mengelola sumber daya komputer dan menjadi perantara antara pengguna, aplikasi, dan perangkat keras.
  • Fungsi utama sistem operasi: manajemen proses, memori, file, I/O, keamanan, dan jaringan.
  • Jenis sistem operasi: Windows, macOS, Linux, Android, iOS.
  • Peran sistem operasi dalam komputasi: booting, menjalankan program, mengakses file, mengelola perangkat.
  • Hubungan dengan arsitektur Von Neumann: OS mengelola CPU, memori, input, output, dan storage.
  • Tanpa sistem operasi, komputer tidak bisa digunakan karena tidak ada antarmuka antara pengguna dan hardware.
Ingat! Sistem operasi adalah "jembatan" yang memungkinkan kita menggunakan komputer. Tanpa OS, komputer hanya kumpulan komponen elektronik yang tidak berguna!
