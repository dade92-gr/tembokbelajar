A. Apa Itu Google Docs?

Google Docs adalah aplikasi pengolah kata berbasis web yang dikembangkan oleh Google. Aplikasi ini memungkinkan pengguna untuk membuat, mengedit, dan menyimpan dokumen secara online serta berkolaborasi dengan pengguna lain secara real-time.

Google: "Google Docs adalah pengolah kata online gratis yang memungkinkan Anda membuat dan memformat dokumen teks, serta berkolaborasi dengan orang lain secara real-time."
Perbedaan Word vs Google Docs:
Microsoft Word (Offline): Install dulu, simpan di PC, kirim via email, edit sendiri
Google Docs (Online): Buka browser, simpan di cloud, share link, bisa barengan

F. Menyisipkan Elemen ke Dokumen

📌 Cara Menyisipkan:
• Gambar: Insert → Image → Pilih sumber (Upload, URL, Google Drive)
• Tabel: Insert → Table → Pilih jumlah baris dan kolom
• Link: Blok teks → Insert → Link (Ctrl+K) → Masukkan URL
• Komentar: Blok teks → Insert → Comment (Ctrl+Alt+M)

G. Fitur Kolaborasi

Cara Membagikan Dokumen:
1. Klik tombol Share di pojok kanan atas
2. Masukkan email orang yang ingin di-share
3. Pilih level akses: Viewer, Commenter, Editor
4. Klik Send

H. Riwayat Revisi

Riwayat Revisi:
File → Version history → See version history
Bisa melihat siapa yang mengubah apa, kapan, dan mengembalikan ke versi sebelumnya.

J. Tips dan Trik

💡 Tips Google Docs:
• Voice Typing: Tools → Voice typing, bisa mengetik dengan suara
• Explore: Tools → Explore, mencari informasi tanpa keluar dokumen
• Dictionary: Klik kanan → Define, mencari arti kata
• Offline Mode: Settings → Offline, bisa mengedit tanpa internet
• Template: File → New → From template, gunakan template siap pakai

K. Rangkuman

  • Google Docs adalah aplikasi pengolah kata online gratis dari Google.
  • Keunggulan: gratis, cloud, kolaborasi real-time, riwayat revisi, akses di mana saja.
  • Fitur dasar: format teks (bold, italic, underline), paragraf, heading, daftar.
  • Menyisipkan elemen: gambar, tabel, link, komentar.
  • Kolaborasi: share dengan viewer/commenter/editor, komentar, mode saran.
  • Riwayat revisi: melihat dan mengembalikan ke versi sebelumnya.
  • Export: bisa diunduh ke berbagai format (Word, PDF, TXT, HTML).
Ingat! Google Docs menyimpan dokumen secara otomatis. Tidak perlu klik save! Pastikan koneksi internet stabil.
