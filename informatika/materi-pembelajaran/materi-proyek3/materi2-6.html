A. Pengantar Logika Pemrograman

Setelah mempelajari HTML dasar dan pseudocode, kini saatnya kita memperdalam logika pemrograman dengan memahami kondisi dan perulangan. Kedua konsep ini adalah fondasi dari hampir semua program yang kompleks.

Ilustrasi Sederhana:
KONDISI (If-Else) seperti mengambil keputusan dalam hidup:
- Jika hujan, maka bawa payung
- Jika tidak, maka tidak perlu bawa payung

PERULANGAN (Looping) seperti melakukan kegiatan berulang:
- Ulangi mencuci piring sampai semua piring bersih
- Ulangi membaca buku sampai halaman terakhir

B. Percabangan (Conditional Statements)

Klik pada setiap jenis percabangan untuk melihat contoh kodenya:

🔹 IF Sederhana
if (kondisi) {
    // Kode jika kondisi true
}
🔹 IF-ELSE
if (kondisi) {
    // Kode jika kondisi true
} else {
    // Kode jika kondisi false
}
🔹 IF-ELSE IF-ELSE (Banyak Kondisi)
if (kondisi1) {
    // Kode jika kondisi1 true
} else if (kondisi2) {
    // Kode jika kondisi2 true
} else {
    // Kode jika semua false
}
📝 Contoh Program Konversi Nilai:
var nilai = 85;
if (nilai >= 90) {
    console.log("Nilai A");
} else if (nilai >= 80) {
    console.log("Nilai B");
} else if (nilai >= 70) {
    console.log("Nilai C");
} else {
    console.log("Nilai D");
}

C. Operator dalam Kondisi

Operator Perbandingan

==
Sama dengan
5 == 5 → true
!=
Tidak sama
5 != 3 → true
>
Lebih besar
5 > 3 → true
<
Lebih kecil
5 < 3 → false
>=
Lebih besar/sama
5 >= 5 → true
<=
Lebih kecil/sama
5 <= 3 → false

Operator Logika

&& (AND)
True jika keduanya true
(5>3) && (3<5) → true
|| (OR)
True jika salah satu true
(5>3) || (3>5) → true
! (NOT)
Membalikkan nilai
!(5>3) → false

D. Perulangan (Looping)

Klik pada setiap jenis perulangan untuk melihat contoh kodenya:

🔄 Perulangan FOR
for (inisialisasi; kondisi; increment) {
    // Kode yang diulang
}

// Contoh: Tampilkan angka 1-5
for (var i = 1; i <= 5; i++) {
    console.log(i);
}
🔄 Perulangan WHILE
while (kondisi) {
    // Kode yang diulang
    // Update variabel
}

// Contoh: Tampilkan angka 1-5
var i = 1;
while (i <= 5) {
    console.log(i);
    i++;
}
🔄 Perulangan DO-WHILE
do {
    // Kode dijalankan minimal sekali
} while (kondisi);

// Contoh: Tampilkan angka 1-5
var i = 1;
do {
    console.log(i);
    i++;
} while (i <= 5);

F. Program dengan Perulangan (Studi Kasus)

📝 Program Tabel Perkalian:
function tampilkan() {
    var a = parseInt(document.getElementById('angka').value);
    var output = "";
    for (var i = 1; i <= 10; i++) {
        output += i + " × " + a + " = " + (i * a) + "<br>";
    }
    document.getElementById('hasil').innerHTML = output;
}

G. Menggabungkan Kondisi dan Perulangan

        console.log(i + " adalah GENAP");

H. Tips Menghindari Infinite Loop

Infinite Loop (Perulangan Tak Berujung) - HINDARI!
Penyebab:
• Lupa menambah increment/decrement
• Kondisi selalu true
• Variabel tidak berubah

Cara Menghindari:
• Pastikan ada increment/decrement
• Pastikan kondisi bisa menjadi false
• Gunakan break jika diperlukan
✅ Contoh Penggunaan break:
for (var i = 1; i <= 10; i++) {
    if (i == 5) {
        break; // Berhenti saat i = 5
    }
    console.log(i);
}
// Output: 1 2 3 4

J. Rangkuman Materi

  • Percabangan (if-else) digunakan untuk pengambilan keputusan berdasarkan kondisi.
  • Operator perbandingan: ==, !=, >, <, >=, <=
  • Operator logika: && (AND), || (OR), ! (NOT)
  • Perulangan FOR digunakan ketika jumlah perulangan sudah diketahui.
  • Perulangan WHILE digunakan ketika perulangan tergantung kondisi.
  • Perulangan DO-WHILE menjamin kode dijalankan minimal satu kali.
  • Gabungan kondisi dan perulangan membuat program lebih powerful.
  • Infinite loop adalah perulangan tak berujung yang harus dihindari.
Ingat! Kondisi dan perulangan adalah jantung dari logika pemrograman. Kuasai konsep ini, dan kamu bisa membuat program apa pun!
