A. Apa Itu Struktur Data?

Struktur data adalah cara menyusun, mengorganisir, dan menyimpan data dalam komputer agar dapat digunakan secara efisien. Struktur data menentukan bagaimana data diakses, ditambahkan, diubah, dan dihapus.

Niklaus Wirth: "Program = Algoritma + Struktur Data" (Program = Algoritma + Struktur Data)
Donald Knuth: "Struktur data adalah hubungan logis antara elemen data."
Ilustrasi Sederhana:
Bayangkan sebuah lemari pakaian:
• Lemari tanpa rak → pakaian menumpuk, susah dicari
• Lemari dengan rak, laci, dan gantungan → pakaian rapi, mudah ditemukan
Struktur data adalah "rak, laci, gantungan" untuk data di komputer.

B. Tipe Data Dasar

Tipe data adalah jenis nilai yang dapat disimpan dan diolah dalam program. Klik pada setiap tipe data untuk memperjelas tampilan (penjelasan tetap terlihat):

Integer
Bilangan Bulat
Contoh: 10, -5, 0, 1000, 2024
Penggunaan: Usia, jumlah, tahun, NIS, jumlah siswa
Operasi: +, -, ×, ÷ (pembagian bulat)
Float / Double
Bilangan Desimal
Contoh: 3.14, 2.5, -0.75, 17.85
Penggunaan: Nilai rata-rata, harga, suhu, tinggi badan
Operasi: +, -, ×, ÷ (pembagian desimal)
String
Teks
Contoh: "Halo", "Informatika", "X TKJ 1", "2024"
Penggunaan: Nama, alamat, teks, keterangan
Operasi: Penggabungan (+), pencarian substring
Boolean
Logika
Contoh: true, false, (5 > 3) → true
Penggunaan: Status (lulus/tidak), kondisi, keputusan
Operasi: AND (&&), OR (||), NOT (!)

C. Variabel: Tempat Menyimpan Data

Variabel adalah tempat penyimpanan data di memori komputer yang memiliki nama, tipe data, dan nilai.

// Deklarasi variabel dalam pseudocode DEKLARASI nama : string usia : integer tinggi : float statusAktif : boolean // Mengisi nilai (assignment) nama ← "Budi" usia ← 16 tinggi ← 165.5 statusAktif ← true

Aturan Penamaan Variabel

✅ Benar: nama, _usia, namaLengkap, jumlahSiswa
❌ Salah: 2nama (awalan angka), nama lengkap (ada spasi), if (kata kunci)
Tips: Gunakan nama variabel yang deskriptif dan mudah dipahami, seperti "jumlahSiswa" bukan "a" atau "b".

D. Tabel Data Sederhana (Array)

Array/Tabel adalah struktur data yang menyimpan kumpulan data dengan tipe yang sama.

Sama dengan
5 == 5 → true
5 == 3 → false

F. Rangkuman

  • Struktur data adalah cara menyusun dan mengorganisir data agar dapat digunakan secara efisien.
  • Tipe data dasar terdiri dari: Integer (bilangan bulat), Float (bilangan desimal), String (teks), Boolean (nilai kebenaran).
  • Variabel adalah tempat penyimpanan data yang memiliki nama, tipe data, dan nilai.
  • Aturan penamaan variabel: diawali huruf/underscore, tanpa spasi, deskriptif.
  • Operator perbandingan: >, <, >=, <=, ==, !=
  • Operator logika: && (AND), || (OR), ! (NOT)
  • Tabel/array digunakan untuk menyimpan kumpulan data dengan tipe yang sama.
  • Pemilihan tipe data yang tepat membuat program lebih efisien dan mudah dipahami.
Ingat! Memilih tipe data yang tepat sangat penting. Menyimpan angka sebagai string akan membuat operasi matematika tidak berjalan dengan benar!
