A. Apa Itu Pseudocode?

Pseudocode adalah cara menuliskan algoritma menggunakan bahasa yang mirip dengan bahasa pemrograman, tetapi masih mudah dipahami oleh manusia. Pseudocode bukan bahasa pemrograman yang bisa dijalankan oleh komputer, melainkan jembatan antara algoritma dan kode program.

Donald Knuth: "Pseudocode adalah representasi informal dari algoritma yang menggunakan konvensi struktural dari bahasa pemrograman."
Posisi Pseudocode dalam Pengembangan Program:
ALGORITMA (Bahasa Manusia) → PSEUDOCODE (Bahasa Semu) → PROGRAM (HTML/JavaScript/Python) → KOMPUTER (Menjalankan program)

B. Mengapa Pseudocode Penting?

📋 Perencanaan
Merancang logika sebelum coding
Kita bisa merancang logika program sebelum menulis kode, sehingga kesalahan logika lebih mudah diperbaiki di awal.
🌐 Tidak Terikat Bahasa
Bisa diterjemahkan ke berbagai bahasa
Pseudocode bisa diterjemahkan ke bahasa pemrograman apa pun (HTML, JavaScript, Python, Java, dll).
⏱️ Hemat Waktu
Kesalahan lebih mudah diperbaiki
Kesalahan logika lebih mudah diperbaiki di pseudocode daripada di kode program yang sudah panjang.
💬 Komunikasi Tim
Memudahkan diskusi antar programmer
Memudahkan diskusi antar programmer karena semua orang bisa memahami pseudocode.
📝 Dokumentasi
Menjadi dokumentasi alur program
Menjadi dokumentasi alur program yang mudah dipahami oleh siapa saja.
📚 Pembelajaran
Membantu pemula memahami logika
Membantu pemula memahami logika pemrograman tanpa harus pusing dengan sintaks bahasa pemrograman.

C. Kata Kunci dalam Pseudocode

Klik pada setiap kata kunci untuk melihat penjelasan lebih detail:

START / BEGIN
Menandai awal algoritma
Digunakan untuk menandai dimulainya eksekusi algoritma. Biasanya dipasangkan dengan END.
END / ENDPROGRAM
Menandai akhir algoritma
Digunakan untuk menandai selesainya eksekusi algoritma.
INPUT / READ
Membaca input dari pengguna
Digunakan untuk menerima data dari pengguna. Contoh: INPUT nama, INPUT usia
OUTPUT / PRINT
Menampilkan output
Digunakan untuk menampilkan hasil ke pengguna. Contoh: OUTPUT "Halo", OUTPUT luas
IF / ELSE / ENDIF
Percabangan kondisi
Digunakan untuk pengambilan keputusan berdasarkan kondisi. Contoh: IF nilai > 75 THEN OUTPUT "Lulus"
FOR / ENDFOR
Perulangan dengan jumlah tetap
Digunakan untuk perulangan yang jumlahnya sudah diketahui. Contoh: FOR i = 1 TO 10
WHILE / ENDWHILE
Perulangan dengan kondisi
Digunakan untuk perulangan yang berhenti berdasarkan kondisi. Contoh: WHILE i < 10
← atau =
Assignment (pemberian nilai)
Digunakan untuk memberi nilai pada variabel. Contoh: luas ← panjang × lebar

D. Struktur Dasar Pseudocode

PROGRAM NamaProgram DEKLARASI // Mendeklarasikan variabel yang digunakan namaVariabel1 : tipeData namaVariabel2 : tipeData ALGORITMA START INPUT data PROSES data OUTPUT hasil END END PROGRAM
📝 Contoh Pseudocode Luas Persegi Panjang:
PROGRAM LuasPersegiPanjang DEKLARASI panjang, lebar, luas : integer ALGORITMA START INPUT panjang INPUT lebar luas ← panjang × lebar OUTPUT "Luas persegi panjang adalah: " + luas END END PROGRAM

E. Contoh Pseudocode untuk Berbagai Kasus

Klik pada setiap contoh untuk melihat pseudocodenya:

🔢 Menentukan Ganjil/Genap
PROGRAM GanjilGenap
DEKLARASI bilangan : integer
ALGORITMA
  START
    INPUT bilangan
    IF bilangan MOD 2 = 0 THEN
      OUTPUT "Genap"
    ELSE
      OUTPUT "Ganjil"
    ENDIF
  END
END PROGRAM
📊 Menghitung Rata-rata
PROGRAM RataTigaBilangan
DEKLARASI a,b,c,jumlah,rata : float
ALGORITMA
  START
    INPUT a, b, c
    jumlah ← a + b + c
    rata ← jumlah / 3
    OUTPUT "Rata-rata: " + rata
  END
END PROGRAM
✅ Program Kelulusan
PROGRAM CekKelulusan
DEKLARASI nilai : float, kkm : float ← 75
ALGORITMA
  START
    INPUT nilai
    IF nilai >= kkm THEN
      OUTPUT "LULUS"
    ELSE
      OUTPUT "TIDAK LULUS"
    ENDIF
  END
END PROGRAM
🔢 Menampilkan Angka 1-10
PROGRAM TampilkanAngka
DEKLARASI i : integer
ALGORITMA
  START
    FOR i = 1 TO 10
      OUTPUT i
    ENDFOR
  END
END PROGRAM

F. Tips Menulis Pseudocode yang Baik

7 Tips Menulis Pseudocode:
1. Gunakan Bahasa Sederhana - Hindari istilah teknis yang rumit
2. Konsisten dengan Kata Kunci - Pakai IF dan ENDIF secara konsisten
3. Beri Indentasi - Gunakan spasi/tab agar struktur terlihat jelas
4. Beri Komentar - Jelaskan bagian yang rumit dengan komentar //
5. Fokus pada Logika - Jangan pusingkan sintaks bahasa pemrograman
6. Uji dengan Algoritma - Pastikan pseudocode menyelesaikan masalah
7. Minta Masukan - Tunjukkan ke teman untuk cek logika

G. Rangkuman

  • Pseudocode adalah cara menulis algoritma dengan bahasa semu yang mirip kode program tapi mudah dipahami manusia.
  • Tujuan pseudocode: memudahkan perencanaan, tidak terikat bahasa, menghemat waktu, komunikasi tim, dokumentasi, dan pembelajaran.
  • Karakteristik pseudocode baik: sederhana, terstruktur, konsisten, tidak ambigu, mudah diterjemahkan.
  • Kata kunci penting: START/END, INPUT/OUTPUT, IF/ELSE/ENDIF, FOR/ENDFOR, WHILE/ENDWHILE.
  • Struktur dasar: PROGRAM → DEKLARASI → ALGORITMA → END PROGRAM.
  • Pseudocode adalah jembatan antara algoritma dan kode program sebenarnya.
Ingat! Pseudocode tidak perlu mengikuti aturan sintaks bahasa pemrograman yang ketat. Yang terpenting adalah logikanya jelas dan mudah dipahami!
