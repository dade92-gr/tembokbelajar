A. Apa Itu HTML?

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat halaman web. HTML bukan bahasa pemrograman, melainkan bahasa markup yang menggunakan tag-tag untuk mendeskripsikan struktur dan konten halaman web.

Tim Berners-Lee (Penemu HTML): "HTML adalah bahasa sederhana yang dirancang untuk membuat hypertext dokumen yang dapat diakses di seluruh dunia."
Ilustrasi Sederhana:
HTML seperti kerangka rumah:
• Tag <html> adalah rumah itu sendiri
• Tag <head> adalah atap rumah (informasi tentang rumah)
• Tag <body> adalah isi rumah (semua yang terlihat)
• Tag <h1> adalah papan nama besar
• Tag <p> adalah ruangan untuk menaruh barang

B. Struktur Dasar Dokumen HTML

<!DOCTYPE html> <html> <head> <title>Judul Halaman</title> </head> <body> <h1>Selamat Datang</h1> <p>Ini adalah paragraf pertama saya.</p> </body> </html>
<!DOCTYPE html>
Deklarasi tipe dokumen
Memberi tahu browser bahwa dokumen ini menggunakan HTML5. Harus ditulis di baris pertama.
<html> ... </html>
Elemen root
Membungkus semua konten HTML. Ini adalah elemen terluar dari halaman web.
<head> ... </head>
Informasi meta
Berisi informasi meta tentang halaman (tidak tampil di browser). Contoh: judul, CSS, meta tag.
<title> ... </title>
Judul halaman
Judul yang tampil di tab browser. Juga digunakan oleh search engine (SEO).
<body> ... </body>
Konten utama
Berisi semua konten yang tampil di halaman web (teks, gambar, video, tombol, dll).

F. Contoh Program Lengkap

Klik pada setiap program untuk melihat kodenya:

📐 Program Luas Persegi Panjang
<h1>Kalkulator Luas</h1>
<label>Panjang:</label>
<input type="number" id="panjang"><br>
<label>Lebar:</label>
<input type="number" id="lebar"><br>
<button onclick="hitungLuas()">Hitung</button>
<p id="hasil"></p>

<script>
function hitungLuas() {
  var p = parseFloat(document.getElementById('panjang').value);
  var l = parseFloat(document.getElementById('lebar').value);
  var luas = p * l;
  document.getElementById('hasil').innerHTML = "Luas: " + luas;
}
</script>
🔢 Program Ganjil/Genap
<h1>Cek Ganjil Genap</h1>
<label>Bilangan:</label>
<input type="number" id="bilangan"><br>
<button onclick="cek()">Cek</button>
<p id="hasil"></p>

<script>
function cek() {
  var bil = parseInt(document.getElementById('bilangan').value);
  if (bil % 2 == 0) {
    hasil.innerHTML = bil + " adalah GENAP";
  } else {
    hasil.innerHTML = bil + " adalah GANJIL";
  }
}
</script>
✅ Program Kelulusan
<h1>Cek Kelulusan</h1>
<label>Nilai:</label>
<input type="number" id="nilai"><br>
<button onclick="cekKelulusan()">Cek</button>
<p id="hasil"></p>

<script>
function cekKelulusan() {
  var nilai = parseFloat(document.getElementById('nilai').value);
  if (nilai >= 75) {
    hasil.innerHTML = "LULUS";
  } else {
    hasil.innerHTML = "TIDAK LULUS";
  }
}
</script>

G. Rangkuman

  • HTML adalah bahasa markup untuk membuat struktur halaman web.
  • Struktur dasar HTML: <!DOCTYPE html>, <html>, <head>, <title>, <body>.
  • Tag dasar: heading (<h1>-<h6>), paragraf (<p>), list (<ul>, <ol>), tabel (<table>), gambar (<img>), link (<a>).
  • Form digunakan untuk menerima input pengguna dengan berbagai tipe: text, number, password, radio, checkbox, select, textarea.
  • JavaScript ditambahkan dengan tag <script> untuk membuat halaman interaktif.
  • Mengambil nilai input: document.getElementById('id').value
  • Struktur kondisi menggunakan if, else if, else.
  • Perulangan menggunakan for dan while.
Ingat! HTML adalah kerangka website. Untuk tampilan yang cantik, pelajari CSS. Untuk interaktivitas yang lebih kompleks, pelajari JavaScript lebih dalam!
