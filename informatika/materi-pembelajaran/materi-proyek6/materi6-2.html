A. Apa Itu Google Sheets?

Google Sheets adalah aplikasi spreadsheet berbasis web yang dikembangkan oleh Google. Aplikasi ini memungkinkan pengguna untuk membuat, mengedit, dan mengolah data dalam bentuk tabel, serta berkolaborasi dengan pengguna lain secara real-time.

Google: "Google Sheets adalah program spreadsheet online yang memungkinkan Anda membuat dan memformat lembar kerja, serta berkolaborasi dengan orang lain secara real-time."
Perbedaan Excel vs Google Sheets:
Microsoft Excel (Offline): Install dulu, simpan di PC, rumus kompleks, Makro/VBA
Google Sheets (Online): Buka browser, simpan di cloud, rumus lengkap, kolaborasi real-time

C. Struktur Dasar Google Sheets

A B C D ┌──────┬──────┬──────┬──────┐ 1│ No │ Nama │ Nilai│ Grade│ ├──────┼──────┼──────┼──────┤ 2│ 1 │ Andi │ 85 │ B │ ├──────┼──────┼──────┼──────┤ 3│ 2 │ Budi │ 92 │ A │ └──────┴──────┴──────┴──────┘ ↑ ↑ ↑ ↑ Kolom A Kolom B Kolom C Kolom D
📌 Istilah Penting:
• Cell (Sel): Pertemuan baris dan kolom, contoh: A1, B2
• Row (Baris): Deretan horizontal, diberi nomor 1, 2, 3
• Column (Kolom): Deretan vertikal, diberi huruf A, B, C
• Range: Kumpulan sel, contoh: A1:C3
• Sheet: Satu halaman spreadsheet
• Workbook: Kumpulan sheet dalam satu file

D. Rumus dan Fungsi Dasar

Klik pada setiap rumus untuk memperjelas tampilan:

=SUM(range)
=SUM(A1:A10)
Menjumlahkan data
=AVERAGE(range)
=AVERAGE(B1:B10)
Rata-rata data
=COUNT(range)
=COUNT(C1:C10)
Menghitung sel berisi angka
=MAX(range)
=MAX(D1:D10)
Nilai tertinggi
=MIN(range)
=MIN(E1:E10)
Nilai terendah
=IF(logika, benar, salah)
=IF(F1>=75,"Lulus","Tidak Lulus")
Percabangan kondisi

F. Jenis Grafik di Google Sheets

Klik pada setiap jenis grafik untuk memperjelas tampilan:

Column Chart
Membandingkan data antar kategori
Line Chart
Menunjukkan tren dari waktu ke waktu
Pie Chart
Proporsi dari total
Bar Chart
Sama dengan column, horizontal
Cara Membuat Grafik:
1. Blok data yang ingin dibuat grafik (termasuk header)
2. Klik menu Insert → Chart
3. Pilih jenis grafik (Column, Line, Pie, Bar)
4. Atur judul, label sumbu, dan warna

G. Fitur Kolaborasi

Cara Membagikan Spreadsheet:
1. Klik tombol Share di pojok kanan atas
2. Masukkan email orang yang ingin di-share
3. Pilih level akses: Viewer, Commenter, Editor
4. Klik Send

I. Rangkuman

  • Google Sheets adalah aplikasi spreadsheet online gratis dari Google.
  • Struktur: Cell (pertemuan baris dan kolom), Row (baris), Column (kolom), Range (kumpulan sel).
  • Rumus dasar: SUM, AVERAGE, COUNT, MAX, MIN, IF.
  • Operator aritmatika: +, -, *, /, ^.
  • Grafik: Column Chart, Line Chart, Pie Chart, Bar Chart.
  • Kolaborasi: Share dengan level akses Viewer, Commenter, Editor.
  • Proteksi: Bisa memproteksi sel/range agar tidak diedit sembarangan.
Ingat! Google Sheets menyimpan data secara otomatis. Pastikan koneksi internet stabil saat bekerja!
