A. Apa Itu Tipe Data?

Tipe data adalah jenis nilai yang dapat disimpan dan diolah dalam program. Pemilihan tipe data yang tepat sangat penting agar program berjalan dengan benar dan efisien.

Niklaus Wirth: "Program = Algoritma + Struktur Data" (Program = Algoritma + Struktur Data)
Ilustrasi Sederhana:
Bayangkan kamu memiliki beberapa kotak untuk menyimpan barang:
• Kotak kecil (Integer) → hanya untuk benda kecil seperti buku
• Kotak sedang (Float) → untuk benda berukuran sedang
• Kotak besar (String) → untuk benda besar
• Kotak khusus (Boolean) → hanya untuk barang ON/OFF
Memilih tipe data = memilih kotak yang tepat!

B. Empat Tipe Data Dasar

Klik pada setiap tipe data untuk memperjelas tampilan:

Integer
Bilangan Bulat
Contoh: 10, -5, 0, 1000, 2024, 85
Penggunaan: Usia, jumlah, tahun, NIS, nilai
⚠️ Hati-hati: 3/2 = 1 (pembagian bulat)
Float
Bilangan Desimal
Contoh: 3.14, 2.5, -0.75, 17.85, 85.5
Penggunaan: Nilai rata-rata, harga, suhu, tinggi
✅ 3/2 = 1.5 (pembagian desimal)
String
Teks
Contoh: "Halo", "Informatika", "X TKJ 1", "2024"
Penggunaan: Nama, alamat, kelas, teks
⚠️ "10" + "5" = "105" (bukan 15!)
Boolean
Logika (Benar/Salah)
Contoh: true, false, (5 > 3) → true
Penggunaan: Status aktif, kondisi, keputusan
✅ Hanya 2 nilai: True atau False

C. Variabel: Tempat Menyimpan Data

Variabel adalah tempat penyimpanan data di memori komputer yang memiliki nama, tipe data, dan nilai.

// Deklarasi variabel dalam pseudocode DEKLARASI nama : string usia : integer tinggi : float statusAktif : boolean // Mengisi nilai (assignment) nama ← "Budi" usia ← 16 tinggi ← 165.5 statusAktif ← true
Tips Penamaan Variabel:
• Gunakan nama yang deskriptif: namaSiswa bukan a
• Awali dengan huruf atau underscore: _usia atau jumlahSiswa
• Hindari spasi dan karakter khusus
• Jangan gunakan kata kunci (if, for, while, dll)

D. Representasi Data dalam Tabel

Tabel adalah cara menyajikan data dalam baris dan kolom. Setiap kolom memiliki tipe data yang sesuai.

Contoh Tabel Data Siswa

NoNamaUsiaKelasNilai Rata-rataAktif
1Andi16X IPA 185.5✅
2Budi17X IPA 278.0✅
3Cici16X IPS 192.3✅
4Dedi17X IPA 167.5❌

Struktur Tabel di Atas:
• No → Integer
• Nama → String
• Usia → Integer
• Kelas → String
• Nilai Rata-rata → Float
• Aktif → Boolean

E. Logika Dasar dengan Boolean

Operator Perbandingan

>
Lebih besar
5 > 3 → true
5 > 7 → false
<
Lebih kecil
5 < 7 → true
5 < 3 → false
>=
Lebih besar atau sama
5 >= 5 → true
5 >= 3 → true
<=
Lebih kecil atau sama
5 <= 5 → true
5 <= 7 → true
==
Sama dengan
5 == 5 → true
5 == 3 → false
!=
Tidak sama
5 != 3 → true
5 != 5 → false

Operator Logika

&& (AND)
Dan
Bernilai true jika keduanya true
(5>3) && (3<5) → true
|| (OR)
Atau
Bernilai true jika salah satu true
(5>3) || (3>5) → true
! (NOT)
Tidak/Balik
Membalikkan nilai
!(5>3) → false
