Keamanan Jaringan Dasar

Proyek 6: Saya Siap Jadi Kreator Digital - Pertemuan 4 4 JP TP 2.5

🎯 Tujuan Pembelajaran

Memahami konsep dasar jaringan komputer
Mengenal jenis-jenis jaringan (kabel dan nirkabel)
Memahami ancaman keamanan pada jaringan
Menerapkan konfigurasi keamanan dasar pada jaringan Wi-Fi
Melindungi perangkat saat terhubung ke jaringan publik

A. Apa Itu Jaringan Komputer?

Jaringan komputer adalah kumpulan dua atau lebih perangkat komputer yang saling terhubung untuk berbagi sumber daya (data, file, printer, koneksi internet) dan berkomunikasi satu sama lain.

Tanenbaum: "Jaringan komputer adalah kumpulan komputer otonom yang saling terhubung."
Ilustrasi Sederhana:
INTERNET → MODEM/ROUTER → KOMPUTER, LAPTOP, HP (saling terhubung dan berbagi koneksi)

B. Jenis-Jenis Jaringan

Klik pada setiap jenis jaringan untuk memperjelas tampilan:

Jaringan Kabel (Wired)
Media: Kabel UTP, fiber optic. Kelebihan: stabil, cepat, aman. Kekurangan: ribet, tidak fleksibel.
Jaringan Nirkabel (Wireless)
Media: Gelombang radio (Wi-Fi). Kelebihan: fleksibel, mobile. Kekurangan: rentan disadap.

C. Ancaman Keamanan Jaringan

Klik pada setiap ancaman untuk memperjelas tampilan:

Eavesdropping
Menyadap lalu lintas data → Data pribadi bisa dicuri
Man-in-the-Middle
Menyusup di tengah komunikasi → Data diubah/dicuri
Rogue Access Point
Access Point palsu → Data masuk ke jaringan palsu
Password Cracking
Membobol password Wi-Fi → Akses gratis ke internet

D. Enkripsi Wi-Fi

Klik pada setiap metode enkripsi untuk memperjelas tampilan:

WEP
Sangat Lemah
Sudah usang, mudah diretas
WPA
Cukup
Pengganti WEP, masih bisa diretas
WPA2
Kuat
Standar saat ini, cukup aman
WPA3
Sangat Kuat
Terbaru, paling aman
📌 Rekomendasi: Gunakan enkripsi WPA2 atau WPA3 untuk keamanan optimal.

E. Tips Keamanan Wi-Fi Rumah

1
Ganti Password Default Router
Password default (admin/admin) harus diganti
2
Gunakan WPA2/WPA3
Pilih enkripsi terkuat yang didukung
3
Password Kuat
Minimal 12 karakter, kombinasi huruf, angka, simbol
4
Nonaktifkan WPS
WPS mudah diretas
5
Aktifkan Firewall
Melindungi dari serangan luar
6
Update Firmware
Perbarui firmware router secara berkala
7
MAC Address Filtering
Hanya perangkat tertentu yang bisa connect

F. Keamanan Wi-Fi Publik

Risiko Wi-Fi Publik:
• Man-in-the-Middle - Data bisa disadap
• Fake Access Point - Jaringan palsu yang mencatat data
• Snooping - Password dan data pribadi bisa dicuri
✅ Tips Aman di Wi-Fi Publik:
• Gunakan VPN (Virtual Private Network)
• Pastikan website menggunakan HTTPS (gembok hijau)
• Jangan akses mobile banking atau akun penting
• Matikan file sharing
• Lupakan jaringan setelah selesai

G. Apa Itu VPN?

TANPA VPN:
ANDA → ISP → WEBSITE
Data terbuka (bisa disadap)

DENGAN VPN:
ANDA → VPN SERVER → WEBSITE
Data terenkripsi (aman)
Manfaat VPN:
• Enkripsi data - Data aman dari penyadapan
• Privasi - Alamat IP asli disembunyikan
• Akses konten - Bisa mengakses konten yang diblokir

H. Checklist Keamanan Wi-Fi

I. Rangkuman

  • Jaringan komputer adalah kumpulan perangkat yang saling terhubung untuk berbagi sumber daya.
  • Jaringan kabel lebih aman tapi tidak fleksibel; jaringan nirkabel (Wi-Fi) fleksibel tapi rentan.
  • Ancaman keamanan: eavesdropping, man-in-the-middle, rogue access point, password cracking.
  • Enkripsi Wi-Fi: WEP (lemah), WPA (cukup), WPA2 (kuat), WPA3 (sangat kuat).
  • Tips keamanan Wi-Fi rumah: ganti password default, pakai WPA2/WPA3, password kuat, nonaktifkan WPS, aktifkan firewall.
  • Wi-Fi publik berbahaya: gunakan VPN, hindari transaksi sensitif, pastikan HTTPS.
  • VPN mengenkripsi data dan menyembunyikan alamat IP.
Ingat! Keamanan jaringan adalah tanggung jawab bersama. Mulai dari diri sendiri dengan mengamankan perangkat dan jaringan di rumah!

🎯 Refleksi Diri

  • Apakah password Wi-Fi-mu sudah kuat?
  • Pernahkah kamu menggunakan Wi-Fi publik? Apa yang kamu rasakan?
  • Apakah kamu akan menggunakan VPN setelah ini?
