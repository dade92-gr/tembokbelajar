A. Pseudocode: Jembatan Menuju Program

Pseudocode adalah cara menuliskan algoritma menggunakan bahasa yang mirip dengan bahasa pemrograman, tetapi masih mudah dipahami oleh manusia. Pseudocode bukan bahasa pemrograman yang bisa dijalankan oleh komputer, melainkan jembatan antara algoritma dan kode program.

Donald Knuth: "Pseudocode adalah representasi informal dari algoritma yang menggunakan konvensi struktural dari bahasa pemrograman."

Kata Kunci dalam Pseudocode

START / END
Menandai awal dan akhir algoritma
INPUT / READ
Membaca input dari pengguna
OUTPUT / PRINT
Menampilkan hasil
IF / ELSE / ENDIF
Percabangan kondisi
← / =
Memberi nilai pada variabel

Contoh Pseudocode

PROGRAM CekBilanganGanjilGenap DEKLARASI bilangan : integer ALGORITMA START OUTPUT "Masukkan bilangan: " INPUT bilangan IF bilangan % 2 == 0 THEN OUTPUT "Bilangan genap" ELSE OUTPUT "Bilangan ganjil" ENDIF END END PROGRAM

B. Apa Itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. HTML digunakan untuk membuat struktur halaman web dan menampilkan konten seperti teks, gambar, video, dan form.

Ilustrasi Sederhana:
HTML seperti kerangka bangunan. HTML menentukan struktur halaman (kepala, badan, footer), sedangkan CSS adalah dekorasi (warna, gaya), dan JavaScript adalah fungsi (interaktivitas).

C. Struktur Dasar HTML

<!DOCTYPE html> <!-- Deklarasi tipe dokumen --> <html> <!-- Elemen root --> <head> <!-- Kepala halaman --> <title>Halaman Saya</title> </head> <body> <!-- Isi halaman --> <h1>Selamat Datang!</h1> <p>Ini paragraf pertama.</p> </body> </html>
<html> ... </html>
Elemen root yang membungkus semua konten halaman
<head> ... </head>
Berisi informasi meta, judul, dan link ke CSS
<body> ... </body>
Berisi konten yang ditampilkan di halaman web

D. Tag-Tag HTML Dasar

<h1> - <h6>
Heading (judul), h1 terbesar, h6 terkecil
<p>
Paragraf (teks biasa)
<br>
Line break (baris baru)
<input>
Input field untuk data pengguna
<button>
Tombol untuk menjalankan aksi
<div>
Container untuk mengelompokkan elemen

E. Program Sederhana dengan HTML + JavaScript

<!DOCTYPE html> <html> <head> <title>Cek Ganjil Genap</title> </head> <body> <h1>Program Cek Bilangan Ganjil Genap</h1> <input type="number" id="bilangan" placeholder="Masukkan bilangan"> <button onclick="cek()">Cek</button> <p id="hasil"></p> <script> function cek() { var bil = document.getElementById('bilangan').value; if (bil % 2 == 0) { document.getElementById('hasil').innerHTML = bil + " adalah bilangan genap"; } else { document.getElementById('hasil').innerHTML = bil + " adalah bilangan ganjil"; } } </script> </body> </html>
📌 Hubungan Pseudocode dengan HTML:
Pseudocode: IF bilangan % 2 == 0 THEN OUTPUT "Genap" ELSE OUTPUT "Ganjil"
HTML + JavaScript: if (bil % 2 == 0) { ... } else { ... }
Ingat! Pseudocode adalah jembatan antara algoritma dan kode program. Tulis pseudocode dengan jelas agar mudah diterjemahkan ke HTML dan JavaScript!
