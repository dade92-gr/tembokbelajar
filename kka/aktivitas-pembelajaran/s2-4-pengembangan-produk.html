Bagian F: Deployment Kelompok 20 menit

💡 Pilih metode deployment sesuai kemampuan:
• Hosting (PHP/MySQL) - Upload ke hosting seperti Niagahoster, Hostinger
• Google Apps Script - Deploy sebagai Web App (✅ HP Friendly, gratis)
• Replit / Glitch - Untuk Node.js / Python (✅ Bisa dari HP)
• Vercel / Netlify - Untuk frontend statis (HTML/CSS/JS)
