💻 Bagian D: Praktek SQL Dasar (Bisa dari HP)

PRAKTEK SQL dari HP - GRATIS & TANPA INSTALASI!

📱 Cara mengakses:
1. Buka Chrome/Safari di HP, ketik sqliteonline.com
2. Pilih menu SQLite (paling ringan untuk HP)
3. Halaman akan terbuka, kamu bisa langsung mengetik perintah SQL
4. Tekan tombol Run (▶) untuk menjalankan perintah
5. Hasil akan langsung muncul di bawah
⚠️ PERHATIAN UNTUK SQLite:
SQLite TIDAK MENGGUNAKAN perintah CREATE DATABASE dan USE seperti MySQL.
Di SQLite, database langsung terbuat saat kita membuat tabel. Jadi LANGSUNG buat tabel tanpa perintah CREATE DATABASE!
📌 IKUTI LANGKAH-LANGKAH BERIKUT (KHUSUS SQLite):

Langkah 1: Buat Tabel (LANGSUNG, tanpa CREATE DATABASE)
CREATE TABLE barang (
  id_barang INTEGER PRIMARY KEY,
  nama_barang TEXT,
  harga INTEGER,
  stok INTEGER
);
Langkah 2: Masukkan Data (INSERT)
INSERT INTO barang VALUES (1, 'Buku Tulis', 5000, 100);
INSERT INTO barang VALUES (2, 'Pensil', 2000, 200);
INSERT INTO barang VALUES (3, 'Penghapus', 1500, 150);
INSERT INTO barang VALUES (4, 'Penggaris', 3000, 80);
Langkah 3: Lihat Data (SELECT)
SELECT * FROM barang;
SELECT * FROM barang WHERE harga > 2000;
Langkah 4: Update Data (Ubah Data)
UPDATE barang SET stok = 90 WHERE nama_barang = 'Buku Tulis';
Langkah 5: Hapus Data (DELETE)
DELETE FROM barang WHERE nama_barang = 'Penghapus';
Langkah 6: Coba SELECT lagi untuk melihat perubahan
SELECT * FROM barang;
1. Apakah kamu berhasil menjalankan perintah SQL? (Ya/Tidak)
2. Tuliskan perintah SQL yang kamu gunakan untuk membuat tabel!
3. Tuliskan perintah SQL untuk menampilkan semua data!
4. Setelah menjalankan SELECT, apa hasil yang tampil? (tulis data yang keluar)
5. Apa yang kamu pelajari dari praktek SQL ini?

📌 LANGKAH PRAKTEK GOOGLE SHEETS:

1. Buka Google Sheets → ketik sheets.new di browser HP.
2. Buat tabel sederhana seperti contoh di bawah (isi minimal 5 baris):

NoNama BarangJumlah TerjualHarga Satuan

3. Hitung total penjualan (Jumlah Terjual × Harga Satuan) pakai rumus.

6. Screenshot hasilnya, upload ke Google Drive, lalu tempel tautannya di bawah.
